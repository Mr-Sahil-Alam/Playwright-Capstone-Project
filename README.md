# Playwright Capstone Project

## Project Overview
This project is an enterprise-grade Playwright automation framework developed as part of the Wipro SDET Playwright Capstone Project.

The framework is designed to automate multiple business services/modules using Playwright and JavaScript with modern automation practices including:
- Page Object Model (POM)
- Data-driven testing
- API testing
- Reporting
- Screenshot capture
- Parallel execution
- Robust locator strategies

---

# Technology Stack

- Playwright
- JavaScript
- Node.js
- Git & GitHub
- Allure Report
- HTML Report
